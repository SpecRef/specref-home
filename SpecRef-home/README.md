# SpecRef Home

Professional material data reference website for quantity surveyors and estimators.

## Features

- Industrial-themed landing page
- Navigation to:
  - Piping specifications
  - Pipe fittings catalog
  - Structural steel data

## Tech Stack

- React
- Tailwind CSS v4
- Vite

## Development

```bash
pnpm install
pnpm run dev
```

## Build

```bash
pnpm run build
```
